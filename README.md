# LinkedIn Solopreneur Skill

A Claude skill for building your LinkedIn personal brand as an n8n/AI Automation expert.
Powered by Justin Welsh's Niche of One philosophy.

## Commands
- `/post [topic]` — Write a LinkedIn post
- `/comment [content]` — Write a strategic comment
- `/outreach [person/goal]` — Write outreach messages
- `/profile [section]` — Optimize profile sections
- `/ideate` — Generate 10 post ideas
- `/newsletter [topic]` — Write a full newsletter

## Files
- `SKILL.md` — Main skill file. Upload this to Claude.
- `references/frameworks.md` — All 21 Justin Welsh strategy frameworks
- `references/templates.md` — Post, outreach, newsletter & profile templates

## Install
1. Go to [claude.ai](https://claude.ai)
2. Open or create a Project
3. Go to Project Files -> Upload
4. Upload `SKILL.md`
5. Claude will now follow the skill automatically in that project
